# Text

A block of text separated by blank spaces.

## Example

```php
use Flowframe\Brief\Components\Html;
use Flowframe\Brief\Components\Body;
use Flowframe\Brief\Components\Text;

Html::make(
    Body::make(
        Text::make('Slot')
    )
);
```
