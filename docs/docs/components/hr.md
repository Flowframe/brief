# Hr

Display a divider that separates content areas in your email.

## Example

```php
use Flowframe\Brief\Components\Html;
use Flowframe\Brief\Components\Body;
use Flowframe\Brief\Components\Hr;

Html::make(
    Body::make(
        Hr::make()
    )
);
```
